Test #1 - Simple Picker

This is supposed to just open a modal window with a simple picker

Click on the button below to see in action and click outside to close the picker

Usage:

$("#button").click(function() {
	$.picker("regular", "my-picker", {});
});

Format:

Picker needs a type (string), an id for the new element (string), and a few extra options (object), as follows:

$.picker([type], [id], [options]);
