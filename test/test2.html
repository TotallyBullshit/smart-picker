Test #2 - Event callbacks

beforeShow

There are similar options for: beforeShow, afterShow, beforeHide, afterHide

Usage:

var picker = new Picker("regular", "my-picker", {
	beforeShow: function() {
		alert("This happens before the picker is created");
	}
});
picker.show();

onInteraction

The callback is called everytime the value changes

The value now is:

Usage:

var picker = new Picker("regular", "my-picker", {
	onInteraction: function(value) {
		$("#result").html(value);
	}
});
picker.show();

onSet

The value now is set to:

Usage:

var picker = new Picker("regular", "my-picker", {
	onSet: function() {
		$("#result").html(value);
	},
	confirmButton: "Confirm this number" //true works fine too
});
picker.show();

Combined: onInteraction and onSet

Current selection:

Usage:

var picker = new Picker("regularList", "my-picker", {
	onInteraction: function(value) {
		$("#result").html(value);
	},
	onSet: function(value) {
		alert("Selected value: "+value);
	},
	confirmButton: "Confirm this number" //true works fine too
});
picker.show();
