Example #1 - Simple smartPicker

This is supposed to just open a modal window with a simple picker

Click on the button below to see in action and click outside to close the picker

Usage:

var my_picker = new smartPicker("regular", "my-picker", {});
$("#button").click(function() {
	my_picker.show();
});

Format:

smartPicker needs a type (string), an id for the new element (string), and a few extra options (object), as follows:

new smartPicker([type], [id], [options]);
