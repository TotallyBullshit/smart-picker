Example #3 - Options

Position and Size

The available positions are: top, left, bottom, right, width and height

Usage:

var picker = new smartPicker("regular", "my-picker", {
	top: 300,
	width: '600px'
});
picker.show();

Transitions

The available transitions are: 'no-transition', 'fade', 'slide', 'slideUp', 'slideRight', 'slideLeft'

The value now is:

Usage:

var picker = new smartPicker("regular", "my-picker", {
	transition: 'fade'
});
picker.show();

Close Button and Confirmation Button

You may overwrite the picker's styles, or you can simply add a class to make it look different

Usage:

var picker = new smartPicker("regular", "my-picker", {
	closeButton: true,
	confirmButton: "Confirm here", //any string or boolean values
});
picker.show();

Click Overlay

If you want to disable clicking the overlay to close the modal, set it to false... It only makes sense for modals.

Usage:

var picker = new smartPicker("regular", "my-picker", {
	clickOverlay: false
});
picker.show();

Drag it!

You can have draggable modals too...

Usage:

var picker = new smartPicker("regular", "my-picker", {
	clickOverlay: false
});
picker.show();
